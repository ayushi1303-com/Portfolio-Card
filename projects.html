Github profile Generator



This code creates a simple **GitHub Profile Viewer** using HTML, CSS, and JavaScript. It includes a text input field where users can enter a GitHub username and a button to generate a profile link. When the button is clicked, JavaScript updates an anchor tag with the GitHub profile URL. The design features a centered layout with a styled button and input field for a clean user experience.

Rock/Paper/Scissor Game



This HTML and CSS code creates a simple Rock-Paper-Scissors game interface. It includes a centered game box with three buttons, each containing an image representing rock, paper, and scissors. Clicking a button triggers JavaScript functions (`rock()`, `scissor()`, `paper()`) to handle game logic (though the JavaScript itself is missing).

Voice-to-Text Generator



This is a simple "Voice to Text" web application using HTML, CSS, and JavaScript. It provides a user-friendly interface with a styled container and a button to start speech recognition. When the button is clicked, the browser's speech recognition API converts spoken words into text and displays them in a textarea.

Text-to-Voice Generator



This is a "Text-to-Voice Converter" web app using HTML, CSS, and JavaScript. Users can enter text into a textarea and click a button to have it read aloud using the browser's SpeechSynthesis API. The interface is simple, with a styled container and interactive button effects.

Password Generator



This is a "Password Generator" web app that creates a random password based on user input (name, last name, and number). It selects random characters from the inputs, adds special symbols, and displays a generated password. The interface includes styled input fields and a button for easy interaction.

Instagram Profile Generator



This is an "Instagram Profile Checker" web app that allows users to enter an Instagram username and generate a direct profile link. The page has a stylish pink-themed design with a rotating Instagram logo. A button labeled "CHECK" updates a clickable link to the entered username’s profile.
